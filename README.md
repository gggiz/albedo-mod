# Albedo - Guardian Overseer

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Minecraft](https://img.shields.io/badge/Minecraft-26.1-blue)]()

Minecraft 26.1 Fabric 模组，将《不死者之王》(Overlord) 中的雅儿贝德作为可召唤的强大随从。

## 召唤方式

将**下界之星**丢到**下界合金块**上，雅儿贝德即被召唤降临。召唤时下界之星和合金块将被消耗。

召唤后**右键**雅儿贝德，她将成为你的随从。

## 随从指令

右键雅儿贝德循环切换三种状态（经验条上方显示当前状态）：

| 状态 | 行为 |
|------|------|
| **跟随** | 跟随玩家移动，距离过远自动瞬移 |
| **巡逻** | 在当前位置游走，自动攻击 12 格内敌对生物 |
| **待机** | 原地静止，不移动不攻击 |

## 随从属性

| 属性 | 值 |
|------|-----|
| 生命值 | 400 |
| 攻击力 | 18 |
| 盔甲 | 40 |
| 移动速度 | 0.32 |
| 击退抗性 | 100% |
| 弹射物免疫 | 是 |

## 战斗能力

随从状态下雅儿贝德会协助你作战，根据当前血量自动切换战斗形态：

| 阶段 | HP | 特性 |
|------|-----|------|
| P1 黑色守护者 | 400–250 | 近战横扫，攻速慢但伤害高 |
| P2 地狱深渊 | 249–130 | 解锁突刺 + 斩击波 |
| P3 魅魔真容 | 129–0 | 装甲碎裂，速度提升，解锁分身 + AOE |

### 技能

1. **横扫千军** — 180° 扇形攻击，16 伤害
2. **地狱突刺** — 直线冲锋 6 格，22 穿甲伤害
3. **深渊斩击波** — 5 道弧形粒子波，每道 12 伤害
4. **暗影分身** — 召唤 2 个幻影，存活 8 秒
5. **终焉裁决** — 跃起砸地，半径 5 格 AOE 25 伤害 + 失明

## 掉落物

雅儿贝德死亡时掉落：

| 物品 | 概率 | 说明 |
|------|------|------|
| 魅魔之角 | 100% | 头盔，+40 护甲，夜视 + 力量 I + 伤害吸收 III |
| 守护者碎片 | 100% | 3–5 个，合成材料 |
| 地狱深渊 | 25% | 戟斧，攻击力 20，横扫范围 +2 |

## 合成配方

- **附魔金苹果**：8 个守护者碎片 + 1 个金苹果 → 64 个附魔金苹果

## 安装

1. 安装 [Fabric Loader](https://fabricmc.net/) 0.18.4+
2. 安装 [Fabric API](https://modrinth.com/mod/fabric-api) 0.145.1+
3. 安装 [GeckoLib](https://modrinth.com/mod/geckolib) 5.5+
4. 下载 [albedo-1.0.0.jar](https://github.com/gggiz/albedo-mod/releases/latest) 放入 `mods` 文件夹

## 命令

```
/summon albedo:albedo
```

## 构建

```bash
./gradlew build
```

要求 Java 25 + Gradle 9.4+。
