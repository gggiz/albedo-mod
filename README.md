# Albedo - Guardian Overseer

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Minecraft](https://img.shields.io/badge/Minecraft-26.1-blue)]()

Minecraft 26.1 Fabric 模组，将《不死者之王》(Overlord) 中的雅儿贝德作为可召唤的强大随从。

## 召唤方式

将**下界之星**丢到**下界合金块**上，雅儿贝德即被召唤降临。召唤时下界之星和合金块将被消耗。

也可使用**雅儿贝德刷怪蛋**直接生成。

召唤后**右键**雅儿贝德，她将成为你的随从。

## 随从指令

右键雅儿贝德循环切换状态（Boss 条上方显示当前状态）：

| 状态 | 行为 |
|------|------|
| **跟随** | 跟随玩家移动，距离过远自动瞬移，协助攻击 |
| **巡逻** | 在当前位置游走，自动攻击 12 格内敌对生物 |
| **待机** | 原地静止，不移动不攻击 |
| **建造模式** | 读取你加载的 Litematica 投影，自动放置方块进行建造 |

### 建造模式使用说明

1. 安装 [Litematica](https://www.curseforge.com/minecraft/mc-mods/litematica) 并在游戏中加载一个投影
2. 右键雅儿贝德切换到**建造模式**（Boss 条显示 `建造中 X%`）
3. 雅儿贝德自动读取当前选中的投影并开始建造
4. Boss 条显示建造进度和放置数量（如 `建造中 50% 3/6`）
5. 建造完成后 Boss 条显示 `等待投影数据...`
6. 要再次建造：切换回建造模式即可（跟随→巡逻→待机→建造）

> **注意**：同一投影重复建造时，已存在的方块不会重复放置，Boss 条会显示 `0/N`。

## AI 聊天

当雅儿贝德在 10 格范围内时，在聊天栏发消息即可与她对话。她以雅儿贝德的身份用中文回复。

### 配置 API Key

游戏内输入命令即可设置（自动保存到配置文件）：

```
/albedokey sk-your-deepseek-api-key
```

输入 `/albedokey` 可查看当前是否已启用。

获取 API Key：[platform.deepseek.com](https://platform.deepseek.com/)

> 未配置 API Key 时聊天功能不会激活，不影响其他功能。

## 战斗能力

随从状态下雅儿贝德会协助你作战，根据当前血量自动切换战斗形态：

| 阶段 | HP | 特性 |
|------|-----|------|
| P1 黑色守护者 | 400–250 | 近战横扫，攻速慢但伤害高 |
| P2 地狱深渊 | 249–130 | 解锁突刺 + 斩击波 |
| P3 魅魔真容 | 129–0 | 装甲碎裂，速度提升，解锁分身 + AOE |

### 技能

1. **横扫千军** — 180° 扇形攻击，16 伤害
2. **地狱突刺** — 直线冲锋 6 格，22 穿甲伤害
3. **深渊斩击波** — 5 道弧形粒子波，每道 12 伤害
4. **暗影分身** — 召唤 2 个幻影，存活 8 秒
5. **终焉裁决** — 跃起砸地，半径 5 格 AOE 25 伤害 + 失明

## EGO

### 以爱与憎之名-大阿卡纳光破斩！！！

源自《边狱巴士》(Limbus Company)，需要长按右键吟唱 18 秒蓄力后释放。

- **蓄力阶段**：面前浮现粉色魔法阵（五芒星 → 六芒星 → 爱心），每 4 秒获得抗性提升 II，每 3 秒地面爆发冲击波
- **释放瞬间**：前方大范围击退 + 虚弱，自身获得伤害吸收 IV（20秒）+ 力量 II（10秒）
- **光束攻击**：60 格射程，持续 5 秒，无视无敌帧造成 **30 魔法伤害**，附带发光 + 虚弱 + 缓慢
- **光束结束后**：获得生命恢复 V，持续 2 秒
- **冷却**：30 秒
- **视觉效果**：暗紫色光束 + END_ROD 外壳粒子 + 末端传送门环 + 90 格冲天光柱

## 安装

1. 安装 [Fabric Loader](https://fabricmc.net/) 0.18.4+
2. 安装 [Fabric API](https://modrinth.com/mod/fabric-api) 0.151.0+
3. （可选）安装 [Litematica](https://www.curseforge.com/minecraft/mc-mods/litematica) 0.27.0+ 以使用建造模式
4. 下载 [albedo-1.0.7.jar](https://github.com/gggiz/albedo-mod/releases/latest) 放入 `mods` 文件夹

## 命令

```
/summon albedo:albedo
```

## 构建（开发者）

```bash
./gradlew build
```

要求 JDK 25 + Gradle 9.5+。普通玩家无需构建，直接下载 Release 即可。
